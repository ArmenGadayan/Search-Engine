ABOUT
This is a basic implementaion of a search engine that uses tf-idf scoring to rank pages

HOW TO RUN

Step 1: Building Index
-run "python3 -m pip install requirements.txt"
-create a DEV/ folder containing all the documents to search
-create a folder called savedData/
-run launch.py

Step 2: Running Search Engine
Terminal: run launch.py to start the command line interface for the search engine
Website: run web.py to start the web browser version of the search engine

-to rebuild the index, delete the contents of savedData/ and run launch.py
